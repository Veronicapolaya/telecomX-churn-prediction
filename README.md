# Telecom X – Predicción de Cancelación de Clientes

## Descripción

Proyecto de Machine Learning desarrollado como parte del Challenge Telecom X del programa Oracle Next Education.

El objetivo del proyecto es construir modelos predictivos capaces de identificar clientes con alta probabilidad de cancelar el servicio (churn), permitiendo a la empresa anticipar estrategias de retención.

## Objetivos

- Preparar datos para modelado
- Analizar correlaciones entre variables
- Entrenar modelos de clasificación
- Evaluar desempeño con métricas
- Identificar factores que influyen en la cancelación

## Tecnologías utilizadas

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Resultado esperado

Un modelo predictivo que permita identificar clientes con mayor riesgo de cancelación y generar insights estratégicos para Telecom X.
